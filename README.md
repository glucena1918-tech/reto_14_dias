Este repositorio contiene el código del proyecto para el reto de 14 días, conectado con Supabase como base de datos y configurado para despliegues automatizados desde GitHub.

- **Base de Datos**: Supabase (PostgreSQL)
- **Control de Versiones**: Git & GitHub
- **Integración Continua**: Sincronización automática de esquemas y migraciones con Supabase GitHub Integration.
